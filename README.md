# 🛒 ShopMart – Modern E-Commerce Web App

A fully responsive and modern **E-Commerce Website** built using **HTML, CSS, and JavaScript**.
This project demonstrates a complete frontend shopping experience with product listing, cart system, and interactive UI.

---

## 🚀 Features

### 🏠 Home Page

* Clean & modern hero section
* Gradient UI with glowing effects
* Call-to-action buttons (Shop Now / Explore)

### 🛍️ Products Page

* Dynamic product listing using JavaScript
* Responsive grid layout:

  * 💻 Desktop → 4 products
  * 📱 Tablet → 2 products
  * 📱 Mobile → 1 product
* Search & category filtering

### 🛒 Cart System

* Add / remove products
* Quantity handling
* Persistent cart using `localStorage`
* Real-time cart count update

### 📞 Contact Page

* Functional contact form
* Success message feedback

### 📱 Fully Responsive

* Mobile-first design
* Hamburger menu navigation

---

## 🧱 Tech Stack

* **HTML5**
* **CSS3 (Advanced UI + Animations)**
* **Vanilla JavaScript (ES6)**

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/shopmart.git
```

2. Open project folder:

```
cd shopmart
```

3. Run in browser:

```
Open index.html
```
